# Companion Module: acCompanimentAlt

This module connects Bitfocus Companion to the acCompanimentAlt Electron application (v1.5.x+).

- WebSocket port: **8877** (default) — configured in acCompanimentAlt App Config
- HTTP remote port: **3000** (default) — used by the web/iPad remote, not Companion

Repository: https://github.com/alanimedia/acCompanimentAlt-companion-module

App repository: https://github.com/alanimedia/acCompanimentAlt
