# Packaged Companion modules

Pre-built `.tgz` files for installing in Companion 4.x via **Modules → Import custom module** (no developer mode required).

| File | Version |
|------|---------|
| [alanimedia-avcueboard-1.10.0.tgz](alanimedia-avcueboard-1.10.0.tgz) | 1.10.0 |

Rebuild with `yarn build` in the repo root (writes `alanimedia-avcueboard-<version>.tgz`, then copy into this folder).
